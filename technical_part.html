Big Data - The place of sport in french society

Presentation

With the multiplication of datas due, inter alia, to IoT, we followed a module of Data Analysis where we first learned to use R which is a programming language for statistical computing and data analysis in order to extract useful datas from enormous datasets. Then, we had a mini-project to apply our new skills: we had to chose an open dataset and lead an analysis on it and summarize it in a report. With Mathieu Raynaud, we chosed to work on the place of sport in the french society.

Resolution of the problem

To determine the place of sport in the french society, we tried to see if the investment, the life expectancy, the migratory tendencies were impacted by the number of sportive equipment per inhabitant. To do so, we analyzed four open data sets, two from the french government (data.gouv.fr) and two from the french national institute of statistic and economic studies (INSEE). Finally, we analyzed these data sets using the programming language R. You can access the GitHub repository in which we stored all the produced algorithms by clicking the button below:

Presentation

During the protocols training, we had to produce a report on the exisiting MAC Layers and more specifically the ones designed for Wireless Sensor Network. Software Defined Radio

Presentation

During the communication course, we had labs about signal defined radio in which we worked on practical applications of wireless communications for Radio and VOLMET messages. In SDR, most of the signal processing is done by software, for this lab, we used GNU Radio. Thanks to this, it's easier to update the system without changing any hardware part and to implement new functionalities. In order to do so, we need to have a receiver that can handle many protocols and a large bandwith, so we use a Universal Software Radio Peripheral (USRP). I worked with Mathieu Raynaud during this project.

Resolution of the problem

Firstly, we did some theoritical work about the IQ modulator, we worked on the mathematical aspect to prove that we could correctly modulate and demodulate signals thanks to this modulator. Then, we moved on the partical part, where we first demodulate radio signals. Thanks to GNU Radio and its various signal processing functions, we could switch between stations and even demodulate in real-time using an USRP. In a second time, we worked on the demodulation of VOLMET messages which is not the same as radio, we implemented our demodulator thanks to GNU radio and it worked with a recording but when we tried to do real-time demodulation, it did not work because our USRP did not cover this bandwith. We, then, summarized our work in a report you can find below.

Knowledge and skills mobilized

For these labs, I mobilized my background skills in signal processing and the skills in telecommunications that I had developped throughout all the Communication course.

Summary and review
